Information

This web application is a work in progress that will act as an interactive resume for me(Dustin Guest). It contains information about me and the skills I have acquired as a software developer.

This web application is developed primarily in javascript and uses the three.js library.

Future Plans

This is where I will describe my future plans for this web application.

  • Improve the structure of the code! It is a bit too procedural and can be improved by a better, more object-oriented design.
  • Improve the look of the subsections, including better, more dynamic transitions when opening the subsections.
  • Add support for mobile devices, including touch controls.
  • Add collision physics to the rocket ship allowing it to interact with objects, and add things for it to interact with (perhaps some asteroids).
  • Add a more dynamic background with twinkling stars.
  • ...
